# EmployeeService
Для запуска проекта откройте терминал в корневой папке проекта и введите следующую команду:
```
docker compose up
```
Далее перейдите по адресу для перехода на клиентскую часть:
```
http://localhost:3000/
```
Для перехода в Swagger Editor:
```
http://localhost:5000/swagger/index.html
```
Задание на знание БД.
Скрипты находятся в папке sql
Написать и приложить скрипты для: 
 - выборки всех сотрудников - файл 1.sql
 - сотрудников у кого зп выше 10000 - файл 2.sql
 - удаления сотрудников старше 70 лет - файл 3.sql
 - обновить зп до 15000  тем сотрудникам, у которых она меньше - файл 4.sql
